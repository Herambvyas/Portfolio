About This Project

TaskMaster Pro is a modern task management application built with vanilla JavaScript, HTML5, and CSS3. It allows users to:

  • Add new tasks
  • Mark tasks as complete/incomplete
  • Edit existing tasks
  • Delete tasks
  • Filter tasks (All/Active/Completed)
  • Clear completed tasks
  • Persist tasks using localStorage

The app features a clean, responsive design that works on both desktop and mobile devices.
